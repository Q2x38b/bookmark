Bookmark URL Parameters

Pipe data into BMarks from browser extensions, Shortcuts, or automation tooling by sending query parameters to /. The capture modal opens with everything pre-filled, even if you still need to sign in.

Quick notes

  • Works with macOS share sheet, Raycast, Alfred, browser extensions, or curl.
  • Links, color values, and plain text are inferred instantly.
  • Unknown payloads become plain text so nothing gets dropped.

Parameters

content required
URL, hex color (e.g. #ff00ff), rgb()/hsl() string, or plain text.
title
Optional custom label. Links auto-fetch the page title; colors use the literal code.
group
Optional group name or UUID. Missing groups are ignored so you can pick one manually.
new
When present (e.g. new=1), the modal opens even if no other fields are sent.

Examples

/?content=https://example.com/docs&title=Docs&group=Design
/?content=%23ffcc00&group=Palette
/?new=1

Behavior

  • Parameters are consumed once and scrubbed from the URL.
  • Not signed in? Payloads are cached for 10 minutes and replayed post-auth.
  • Invalid URLs or color strings fall back to plain text and will need a manual title.
